SchoolHub

HTML/CSS

SchoolHub is a static web app that I've built using only HTML and CSS. It's both mobile and desktop responsive. I've built to present as my HTML/CSS assessment(capstone) project. I followed the requirements and the design specifications given by the assessment board of Microverse School.

The main goal SchoolHub app is to give the simplest means to both parents and guardians to search for the best basic schools for their wards to attend. Users can search for their target schools based on different categories. These include the geographical location of the school, the kind of curriculum the school is running, the accepted gender and much more. School proprietors can list their schools with the schools' profiles for easy identification.

Twitter Redesign

HTML/CSS Ruby on Rails

This web app is a desktop version of the popular social media network Twitter. It is an assessment(capstone) project that I've built to demonstrate mastery in the Ruby on Rails module of the Microverse curriculum after completing that particular module.

The requirements and the design specification were given by the assessment team of Microverse. The user logs in to the app, only by typing the username (a complex authenticated login is not a requirement). After the user has successfully logged in, he or she has can use a simple form to write and post a tweet. Every logged in user can view other users' profile and follow them. You can also stop following any user you previously follow. Your feed section always get updated by the posts of all the users you are following including yours(sorted by most-recent).

Private Events

HTML/CSS Ruby on Rails Bootstrap

Private Events is a responsive web app that people can look for their favorite events to attend. Event organizers can also register their events for people to attend

The app has a simple registration process. All you have to provide is a username and a valid address then you are good to go. User only log in with their unique username. No password is required.

On the home page, all events are categorized by upcoming events and previous events. User can also view all events attended or all events created by him or her.
